# CodeAlpha_HangmanGame(python)
#Description
This is a simple text-based Hangman game written in python.
The player tries to guess a hidden word by entring a single letter at a time.
The game displays the correctly guessed letters and hides the remaining letters with underscores.
##Features
Guess the word letter by letter.
Shows correct and incorrect guesses.
Limited attempts.
Simple command-line interface.
##Requirements
Python 3 installed on the computer.
##How to run the game
Download or copy the python file on your computer.
Open your IDE and open the Hangman.py file.
Click the RUN button and the game will start on terminal
